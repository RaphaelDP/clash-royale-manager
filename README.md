# Clash Royale Clan Manager Dashboard

A dashboard to manage and analyze Clash Royale clan data, including member activity, war performance, and promotion scores.

## Features
- Track member activity and inactivity
- Monitor war performance and participation
- Calculate promotion and kick scores
- Visualize trends with Streamlit
